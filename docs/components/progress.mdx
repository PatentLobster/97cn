# Progress

A progress bar component that replicates the Windows 95 progress bar styling with blue fill.

## Installation

<InstallationTabs 
  componentName="progress" 
  dependencies={["@radix-ui/react-progress"]}
/>

## Usage

<AutoPreview>
  <Win95Progress value={33} className="w-64" />
</AutoPreview>

```tsx
import { Win95Progress } from "@/components/97cn/progress"

export function Example() {
  return <Win95Progress value={33} className="w-64" />
}
```
