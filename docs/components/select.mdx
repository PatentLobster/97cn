# Select

A select dropdown component styled to match Windows 95's dropdown appearance with raised button styling.

  componentName="select"

## Usage

<ComponentPreview>
  <Win95Select>
    <Win95SelectTrigger className="w-48">
      <Win95SelectValue placeholder="Select an option" />
    </Win95SelectTrigger>
    <Win95SelectContent>
      <Win95SelectItem value="option1">Option 1</Win95SelectItem>
      <Win95SelectItem value="option2">Option 2</Win95SelectItem>

## Props

### Win95Select

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| value | string | - | Controlled value |
| defaultValue | string | - | Default value |
| onValueChange | (value: string) => void | - | Callback when value changes |
| disabled | boolean | false | Whether the select is disabled |

### Win95SelectTrigger

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| className | string | - | Additional CSS classes |

### Win95SelectItem

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| value | string | - | Item value |
| disabled | boolean | false | Whether the item is disabled |
