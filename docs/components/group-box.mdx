# Group Box

A group box component that creates a sunken container with an optional label, mimicking Windows 95's fieldset styling.

## Installation

<InstallationTabs componentName="group-box" dependencies={[]} />

## Usage

<ComponentPreview>
  <Win95GroupBox label="Settings" className="w-64">
    <div className="space-y-2 p-2">
      <div className="flex items-center space-x-2">
        <Win95Checkbox id="option1" />
        <label htmlFor="option1" className="text-xs text-win95-black">
          Enable feature A
        </label>
      </div>
      <div className="flex items-center space-x-2">
        <Win95Checkbox id="option2" />
        <label htmlFor="option2" className="text-xs text-win95-black">
          Enable feature B
        </label>
      </div>
      <Win95Button size="sm">Apply</Win95Button>
    </div>
  </Win95GroupBox>
</ComponentPreview>

```tsx
import { Win95GroupBox } from "@/components/97cn/group-box";

export function Example() {
  return (
    <Win95GroupBox label="Settings">
      <div className="p-2">{/* Group content */}</div>
    </Win95GroupBox>
  );
}
```

## Props

| Prop      | Type            | Default | Description                       |
| --------- | --------------- | ------- | --------------------------------- |
| label     | string          | -       | Optional label text for the group |
| className | string          | -       | Additional CSS classes            |
| children  | React.ReactNode | -       | Group content                     |
