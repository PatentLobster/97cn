# Window

A window container component that replicates the classic Windows 95 window appearance with title bar and controls.

## Installation

<InstallationTabs 
  componentName="window" 
  dependencies={["lucide-react"]}
/>

## Usage

<AutoPreview>
  <Win95Window title="My Application" className="w-80 h-48">
    <div className="p-2">
      <p className="text-xs text-win95-black mb-2">This is a Windows 95 style window.</p>
      <Win95Button>OK</Win95Button>
    </div>
  </Win95Window>
</AutoPreview>

```tsx
import { Win95Window } from "@/components/97cn/window"

export function Example() {
  return (
    <Win95Window title="My Application">
      <div className="p-2">
        <p>Window content goes here</p>
      </div>
    </Win95Window>
  )
}
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| title | string | "Window" | Window title text |
| active | boolean | true | Whether window appears active |
| resizable | boolean | true | Show resize handle |
| minimizable | boolean | true | Show minimize button |
| maximizable | boolean | true | Show maximize button |
| closable | boolean | true | Show close button |
| onClose | () => void | - | Close button callback |
| onMinimize | () => void | - | Minimize button callback |
| onMaximize | () => void | - | Maximize button callback |
| statusBar | React.ReactNode | - | Status bar content |
