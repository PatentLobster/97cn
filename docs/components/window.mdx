# Window

A window container component that replicates the classic Windows 95 window appearance with title bar and controls.

## Installation

<InstallationTabs 
  componentName="window" 
  dependencies={["lucide-react"]}
/>

## Usage

<AutoPreview>
  <Win95Window title="My Application" className="w-80 h-48">
    <div className="p-2">
      <p className="text-xs text-win95-black mb-2">This is a Windows 95 style window.</p>
      <Win95Button>OK</Win95Button>
    </div>
  </Win95Window>
</AutoPreview>

## Examples

### Basic Window

<AutoPreview>
  <Win95Window title="Notepad" className="w-72 h-40">

### Window with Custom Controls

<AutoPreview>
  <Win95Window 
    title="Calculator" 
    className="w-64 h-32"

  </Win95Window>
