# Checkbox

A checkbox component that replicates the Windows 95 checkbox styling with sunken appearance.

## Installation

<InstallationTabs 
  componentName="checkbox" 
  dependencies={["@radix-ui/react-checkbox"]}
/>

## Usage

<AutoPreview>
  <div className="flex items-center space-x-2">
    <Win95Checkbox id="terms" />
    <label htmlFor="terms" className="text-xs text-win95-black">
      Accept terms and conditions
    </label>
  </div>
</AutoPreview>

```tsx
import { Win95Checkbox } from "@/components/97cn/checkbox"

export function Example() {
  return (
    <div className="flex items-center space-x-2">
      <Win95Checkbox id="terms" />
      <label htmlFor="terms">Accept terms and conditions</label>
    </div>
  )
}
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| checked | boolean | - | Controlled checked state |
| defaultChecked | boolean | false | Default checked state |
| disabled | boolean | false | Whether the checkbox is disabled |
| onCheckedChange | (checked: boolean) => void | - | Callback when checked state changes |
