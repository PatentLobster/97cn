# Checkbox

A checkbox component that replicates the Windows 95 checkbox styling with sunken appearance.

## Installation

<InstallationTabs 
  componentName="checkbox" 
  dependencies={["@radix-ui/react-checkbox"]}
/>

## Usage

<AutoPreview>
  <div className="flex items-center space-x-2">
    <Win95Checkbox id="terms" />
    <label htmlFor="terms" className="text-xs text-win95-black">
      Accept terms and conditions
    </label>
  </div>
</AutoPreview>
