# Textarea

A multi-line text input component styled to match Windows 95's sunken text area appearance.

## Installation

<InstallationTabs 
  componentName="textarea" 
  dependencies={[]}
/>

## Usage

<ComponentPreview>
  <Win95Textarea placeholder="Enter your message..." className="w-64 h-24" />
</ComponentPreview>

```tsx
import { Win95Textarea } from "@/components/97cn/textarea"

export function Example() {
  return <Win95Textarea placeholder="Enter your message..." />
}
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| placeholder | string | - | Placeholder text |
| disabled | boolean | false | Whether the textarea is disabled |
| value | string | - | Controlled value |
| defaultValue | string | - | Default value |
| rows | number | - | Number of visible text lines |
| cols | number | - | Visible width of the text control |
